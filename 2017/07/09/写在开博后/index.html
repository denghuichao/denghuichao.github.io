开博第一天

2017-07-09

最近几个月一直在看技术书籍，看的数量也不少，但是一直没有做笔记的习惯。最近思考觉得自己看过的东西，还是应该做点记录，于是昨天开了个博客，以后看的书就用它来做做总结。希望这个博客能够记录我技术成长的每一步。

最近一段时间，准备把Java I/O 和 NIO、数据库方面的知识系统的学习一下。准备看看这几本书：

  • JDBC Recipes A Problem Solution Approach
  • Java Database Bible
  • 《Jvava I/O》以及《Jvava NIO》

另外，系统的学一学Servlet、JSP方面的东西。在做web开发，但是这些东西出来没有仔细的看一遍，上来就使用各种框架，很多基础的东西都了解的不够。主要打算看看这几本书：

  • Patterns of Enterprise Application Architecture
  • J2EE PATTERNS SL500
  • Holub on patterns
