开博第一天

2017-07-09

最近几个月一直在看技术书籍，看的数量也不少，但是一直没有做笔记的习惯。最近思考觉得自己看过的东西，还是应该做点记录，于是昨天开了个博客，以后看的书就用它来做做总结。希望这个博客能够记录我技术成长的每一步。

最近一段时间，准备把Java I/O 和 NIO、数据库方面的知识系统的学习一下。准备看看这几本书：

  • JDBC Recipes A Problem Solution Approach
  • Java Database Bible
  • 《Jvava I/O》以及《Jvava NIO》

另外，系统的学一学Servlet、JSP方面的东西。在做web开发，但是这些东西出来没有仔细的看一遍，上来就使用各种框架，很多基础的东西都了解的不够。主要打算看看这几本书：

  • Patterns of Enterprise Application Architecture
  • J2EE PATTERNS SL500
  • Holub on patterns

用Java写个爬虫IP代理池

2017-07-08

简介

最近在玩爬虫, 很多网站有反爬虫机制,会对访问频率太高的IP进行禁封.于是到网上去找免费的代理,使用过程中发现这些代理的质量不高,很多都不能使用,因此想到做个代理池.程序主动收集代理IP,并定时对这些代理进行可用性验证,同时提供接口获取可用的代理供爬虫使用.

代理IP来源

有很多网站提供代理IP服务,有的免费,质量好的则付费.这里只收集免费的代理IP.目前爬取了goubanjia,kuaidaili,66IP,xicidaili这几个网站的代理IP.每个网站IP只爬取了前面10页,后面的基本上不可用.

  • Java
  • Proxy Pool
